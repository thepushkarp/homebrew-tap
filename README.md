# homebrew-tap

Homebrew formulae for [mls](https://github.com/thepushkarp/mls) — terminal-native audio/video/image file browser.

## Install

```bash
brew install thepushkarp/tap/mls
```

This formula installs `ffmpeg` and `mpv`, so metadata probing and playback work
on a fresh Brew install. Install `trash` separately if you want safe delete in
triage mode:

```bash
brew install trash
```

## Update

```bash
brew upgrade mls
```
